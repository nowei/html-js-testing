Testing different buttons

This is a button with text on it

and you get this by doing something like: 

<button>text<\button>

You can style the button so that it's larger

by doing something like this:

<button style="width:10em; height:10em;">text<\button>

and you can change the color like this:

<button style="background-color:yellow">text<\button>

And then you can add animations to the buttons using css and javascript and have it repeat on click but it looks pretty ugly

css:

@keyframes colors {
  from {background-color: red;}
  to {background-color: yellow;}
}
pre {
  text-align: left;
  padding-left: 20%;
}

html:

<button id="coloranimation" class="coloranimation" 
   onclick="var element=document.getElementById('coloranimation'); 
            element.classList.remove('coloranimation'); 
            void element.offsetWidth; 
            element.classList.add('coloranimation');">text<\button>

Then you can try adding some stuff like hover animations

css:

#hovercolor {
  background-color:#000000;
  color: white;
  transition-duration: 0.4s;
}
#hovercolor:hover {
  background-color:#FFFFFF;
  box-shadow: 0 12px 16px 0 rgba(0,0,0,0.24), 0 17px 50px 0 rgba(255,255,255,1);
  color:black;
  transform: scale(2.0);
}

html:

<button id="hovercolor">text<\button>
